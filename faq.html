Frequently Asked Questions

Everything you need to know about Roblox executors, safety, and how things work.

What is a Roblox executor? ▼

A Roblox executor is a third-party tool that allows you to run Lua scripts within Roblox. They inject code into the Roblox client to execute custom scripts that can automate tasks, modify gameplay, or add new features not normally available.

Are Roblox executors safe? ▼

It depends on the executor. Reputable executors from official sources are generally safe, but there are many fake versions that contain malware. Always download from official sites listed on our directory. Never download from random YouTube videos or untrusted links.

What is an UNC score? ▼

The UNC (Universal Naming Convention) test measures an executor's compatibility with standard Roblox Lua functions. A higher UNC score means better script compatibility. sUNC (Strict UNC) is a more rigorous test that checks additional functionality. Scores above 90% are considered excellent.

What does "keyless" mean? ▼

A keyless executor does not require you to obtain a key (usually through linkvertise or similar ad-links) before using it. Keyless executors are more convenient as you can launch and use them immediately without completing any tasks or watching ads.

What is Byfron? ▼

Byfron (now called Hyperion) is Roblox's anti-tamper technology that makes it harder for executors to inject code. It was introduced to combat exploiting. Many modern executors have found ways to bypass it, but it has made executor development more challenging.

Can I get banned for using executors? ▼

Yes, using executors violates Roblox's Terms of Service and can result in account termination. While some users have used executors for extended periods without being banned, there is always a risk. Use executors on alt accounts at your own discretion.

What is the difference between internal and external executors? ▼

Internal executors inject a DLL into the Roblox process and run scripts from within. They are generally more powerful and have better performance. External executors run outside the Roblox process and interact with it through memory reading/writing. External executors are often considered more stable after Roblox updates.

Are there executors for mobile (Android/iOS)? ▼

Yes, there are executors for Android devices. Delta and Codex are popular options listed in our directory. iOS executors are less common due to Apple's stricter security measures.
