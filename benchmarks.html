ⓘ About These Benchmarks

Scores are collected from community testing and official API reports. They reflect real-world performance but may vary by system configuration. UNC/sUNC scores indicate how well an executor supports script functions — higher is better for script compatibility. Injection speed measures the time from launch to script-ready state. Memory usage is the baseline footprint when idle.

Executors marked as Down have no current benchmark data. All data is updated regularly as new versions release.
