Executor Guides

Step-by-step tutorials for downloading, installing, and using each executor safely.

Xeno

Windows
1
Download — Go to xeno.now and download the latest version.
2
Extract — Right-click the downloaded ZIP and select "Extract All". Use password if prompted (check the site).
3
Run — Open the extracted folder and run Xeno.exe as Administrator.
4
Inject — Launch Roblox, then click "Inject" in Xeno. Once injected, paste your script and click "Execute".
Xeno is keyless and external — no keys or DLL injection needed.

Solara

Windows
1
Download — Go to getsolara.dev and download the installer.
2
Install — Run the installer and follow the on-screen instructions.
3
Launch — Open Solara from your desktop shortcut. It will auto-update if needed.
4
Attach & Execute — Launch Roblox, click "Attach" in Solara, then paste and execute your script.
Solara is external and keyless. It was the first executor to bypass Byfron.

Velocity

Windows
1
Download — Go to getvelocity.llc and download the latest build.
2
Extract — Unzip the downloaded file to a folder of your choice.
3
Launch — Run the executable as Administrator.
4
Inject — Launch Roblox, click "Inject" in Velocity, then execute your scripts. Velocity uses internal DLL injection for high performance.
Velocity boasts a 99% sUNC score — one of the highest compatibility ratings available.

Madium

Windows

Real has a free tier with key ads and a paid tier for ad-free premium access.

Delta

Android
1
Download — Go to deltaexploits.gg and download the Delta APK.

Delta is the most popular Android executor with 100% UNC. It requires a free key through linkvertise.

Safety Tips

⚠
Always download from official sources — Only use the links listed on our Executors page. Never download from YouTube descriptions or random Discord DMs.
⚠
Use an alt account — Executors violate Roblox ToS. Always use a secondary account to avoid losing your main account.
⚠
Disable antivirus temporarily — Many executors are detected as "hack tools" by antivirus software. Add an exception rather than disabling permanently.
⚠
Keep executors updated — Roblox updates frequently. Check our update badges to see which executors are still current.
⚠
Run as Administrator — Most Windows executors need admin privileges to inject into Roblox properly.
